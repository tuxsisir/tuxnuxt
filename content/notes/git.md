### git checkout with remote branch updates
```sh
$ git remote update
$ git fetch
$ git checkout --track origin/<BRANCH-NAME>
```

### delete remote branch
```sh
$ git push <remote_name> --delete <branch_name>
```

### git untrack file or a folder

```sh
$ git rm -r --cached path_to_your_folder/
```
